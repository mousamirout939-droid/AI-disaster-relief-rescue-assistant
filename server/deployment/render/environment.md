# Required Render Environment Variables

| Variable | Description |
|---|---|
| `MONGODB_URI` | MongoDB Atlas connection string |
| `MONGODB_DB_NAME` | Database name |
| `JWT_SECRET_KEY` | Long random secret for signing JWTs |
| `GEMINI_API_KEY` | Google Gemini API key (chatbot) |
| `GOOGLE_MAPS_API_KEY` | Google Maps API key (safe routing) |
| `OPENWEATHER_API_KEY` | OpenWeatherMap API key (weather alerts) |
| `CORS_ORIGINS` | JSON array of allowed frontend origins |
