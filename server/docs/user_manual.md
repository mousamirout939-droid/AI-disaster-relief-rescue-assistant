# User Manual

## For affected individuals
- **Report a disaster**: open the Report page, describe what's happening, drop a pin
  or use your current location, and optionally attach a photo for AI severity analysis.
- **SOS**: tap the SOS button for immediate help — this notifies the nearest available
  rescue team and surfaces the nearest hospital.
- **Find shelters/hospitals**: the map view shows the closest open shelters and
  hospitals with live capacity/bed information.
- **Emergency chatbot**: ask the in-app assistant for safety guidance in your preferred
  language.
- **Emergency contacts**: add trusted contacts who should be notified in an emergency.

## For volunteers
Register via the Volunteer page; an admin will review and approve your registration.

## For admins
The Admin Dashboard shows pending reports, active disasters, shelter/hospital capacity,
and rescue team status. Verify or reject reports, dispatch rescue teams, publish alerts,
and manage user accounts.
